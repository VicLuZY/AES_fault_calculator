# Web workstation

`index.html` loads `faultcalc.wasm` from the same directory. Use it through the built-in local server:

```bash
cargo run -p faultcalc-cli -- serve web 8080
```

`faultcalc_workstation.html` is generated by embedding the WASM binary into `index.template.html`:

```bash
cargo run -p faultcalc-cli -- embed-wasm web/index.template.html web/faultcalc.wasm web/faultcalc_workstation.html
```

The generated standalone file can be opened directly from disk and does not need a server.
